    HTML笔记

    ​ HTML 是一种相当简单的、由不同元素组成的标记语言，它可以应用于文本片段，使文本在文档中具有不同的含义（它是一个段落吗？它是一个项目列表吗？它是一个表格吗？），将文档结构化为逻辑块（文档是否有头部？有三列内容？有一个导航菜单？），并且可以将图片，影像等内容嵌入到页面中。本模块将介绍前两个，并且介绍一些理解HTML所需的基本概念和语法。

    如上所述html可以概括三点为

    • HyperText Markup Language（超文本标记语言）
    • 非编程语言
    • 告知浏览器来构建网页

    HTML元素的一般结构

    HTML一段组成结构包括

    • opening tag (起始标签）
      例如<kbd>,<p>等，相当于告诉浏览器生成什么东西
    • content（内容）
      在tag中包裹的内容
    • closing tag（结束标签）
      例如</kbd>,</p>等，结束标记包裹内容
    • Html Element（html元素）
      由tag和content一起就组成了html中的一个最基本元素

    块级元素和内联元素|Inline and Block Level Element

    块级元素 | Block Element
    • 在页面上以块的方式展现
    • 强制出现在新的一行之中
    • 占全部宽度
    内联元素 | Inline Element
    • 通常在块级元素内
    • 不会导致文本换行
    • 只占必要的部分宽度

    HTML属性|attribute

    HTML属性是赋予元素一些功能，例如超链接等等，属性分为简单两种

    • 全局属性|globe attribute
      全局属性是适用于大部分标签的属性
    • 特定属性|attribute
      特定属性只适用于特定标签
