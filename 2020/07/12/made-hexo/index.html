    #hexo#个人博客#博客搭建
    　　这是我搭建自己的博客后，记录自己学习进度的第一篇笔记，这个博客是为了能在今后学习python，html5，css所搭建的，零基础的开始，希望自己能够坚持下来
    　　hexo是一款非常适合搭建博客的工具，简单，快捷，方便易于上手 　　

    准备工作

    第一步安装node.js和git

    使用hexo首先我们需要安装node.js，git，hexo是基于node.js的静态博客搭建器

    node.js

    Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。 Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型。Node js是一个让 JavaScript 运行在服务端的开发平台，它让 JavaScript 成为与PHP、Python、Perl、Ruby 等服务端语言平起平坐的脚本语言　

    git

    Git是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理

    这里我用manjaro来演示如何安装，使用archlinuxcn源仅用两行命令即可，macos以及Windows环境下从链接中下载安装包即可完成安装，linux环境下也可以。下面开始演示

    第二步利用node.js来安装hexo

    因为国内使用npm源来安装会非常慢，所以我们首先利用npm来安装cnpm镜像源，因为cnpm是指向国内淘宝，所以速度会快很多

    npm

    npm 是 Node.js 的包管理工具，用来安装各种 Node.js 的扩展。 npm 是 JavaScript 的包管理工具，也是世界上最大的软件注册表

    下面我们打开konsole终端(快捷键是Ctrl+Shift+T)输入安装指令

    建立一个github仓库，命名必须为 username（你的github名称）.github.io，这是为了开启你的github pages功能以展示你博客的样貌。
